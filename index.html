Python Einstieg

Einleitung

Die Programmiersprache Python erfreut sich zunehmender Beliebtheit. Bekannt wurde die bereits in den 90er Jahren entwickelte Sprache insbesondere im Zusammenhang mit dem Raspbery PI (das „PI“ steht für „Python Interpreter“) sowie durch die zunehmen- de Bedeutung der Sprache im Bereich Machine Learning. Die Pakete Pandas, NumPy, Scikit-learn und TensorFlow erweitern den Sprachumfang von Python für diesen Ein- satzbereich erheblich. Der Name der Programmiersprache bezieht sich auf die englische Komikergruppe Mon- ty Python, die insbesondere in den 70er Jahren durch die Fernsehserie Flying Circus oder auch durch Kinofilme wie Das Leben des Brian, Ritter der Kokosnuss oder Sinn des Le- bens bekannt wurden1. Python ist vor allem aufgrund eines gut lesbaren und strukturierten Quellcode bei vielen Entwicklern beliebt. Im Gegensatz z.B. zu C oder Java verzichtet Python zum Beispiel auf die Verwendung von geschweiften Klammern, um Blöcke zu definieren. Statt- dessen wird in Python auf Einrückungen zurückgegriffen. Das zwingt den Programmierer zu einer übersichtlichen Formatierung des Codes. Die aktuelle Version, die auch in diesem Skript verwendet wird, ist Python 3. Diese Version ist nicht abwärtskompatibel zu älteren Versionen! Konkret wird in diesem Skript die Version 3.8 verwendet. >>> BILD In Python werden Datentypen dynamisch zugewiesen (dynamische Typisierung). Eine explizite Typisierung wie in Java gibt es somit nicht. Python unterstützt die Objektorientierung, erlaubt aber auch die „klassische“ pro- zedurale Programmierung. Sie wird deshalb auch als eine Multiparadigmensprache be- zeichnet, was bisweilen auch für Kritik an der Sprache sorgt. Anders als Java zwingt Python den Programmierer also nicht, objektorientiert zu programmieren. Dieses Skript stellt die wichtigsten Grundkonzepte der Sprache vor, kann aber wie üb- lich nicht auf alle Aspekte der Sprache eingehen. Ich verweise auf die vielfältige Literatur. Einige Buchtipps finden Sie am Ende des Skripts. Dieses Skript soll auch keine Einführung in das allgemeine Programmieren darstellen. Es werden also bereits Grundkenntnisse wie das Verwenden von Verzweigungen, Schleifen etc. vorausgesetzt. Entwicklungsumgebung

Es gibt viele IDEs1für Python, z.B. Sublime, PyCharm, Visual Studio Code oder auch Id- le (übrigens eine Referenz auf Eric Idle, einem Mitglied von Monty Python). Im Rahmen der Schulung werden wir die IDE Spyder verwenden. Sie ist bereits Bestandteil von Ana- conda, das als Installationspaket auf unseren Schulungsrechnern im Softwarecenter zur Verfügung steht. Falls man Anaconda direkt installieren möchte, kann das Installations- programm auf https://www.anaconda.com heruntergeladen werden. Für die Installation benötigt man allerdings lokale Administrationsrechte. Nach der Installation und dem Start von Anaconda kann auch Spyder gestartet werden, indem man auf Launch klickt (siehe Abb. 2.1). >>> BILD Spyder startet man entweder durch Klick auf die Kachel in der Anaconda-Umgebung, über das Startmenü oder direkt in der Kommandozeile durch Eingabe von spyder. Auf Windows-PCs startet man hierzu die Anaconda Powershell Prompt und gibt dort spyder ein und drückt die Enter-Taste. Auf Linux- oder Mac-Systemen öffnet man ein Terminal- Fenster und verfährt dort analog. Nach dem Starten von Spyder sieht die Oberfläche in etwa so aus, wie in Abbildung 2.2 gezeigt. >>> BILD In dem linken Fenster wird der Python-Quellcode eingegeben. Im rechten unteren Fenster sehen wir die Konsole, wo die Ausgaben unseres Programms erfolgen oder wo wir auch einzelne Python-Befehle direkt eingeben können. Installiert man nicht Anaconda sondern ausschließlich Python, dem steht nach der Installation die IDE Idle2 zur Verfügung (Abb. 2.3). In Idle auf File → New klicken, den Python-Quellcode eingeben, Datei speichern und die Taste F5 drücken, um die Datei zu kompilieren und auszuführen. >>> BILD
